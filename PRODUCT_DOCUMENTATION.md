## üìã Product Overview

**UAE Legal Research Platform** is an AI-powered legal research assistant that provides instant answers to UAE legal questions with verified citations from a comprehensive database of 50,000+ legal documents.

### Target Market
- **Individual Users**: Professionals, entrepreneurs, expatriates needing legal guidance
- **SMEs**: Small and medium enterprises requiring regular legal research
- **Legal Professionals**: Lawyers, paralegals, legal consultants
- **Corporate Teams**: In-house legal departments

### Unique Value Proposition
- Instant AI-powered legal answers in natural language
- Verified citations from 50,000+ UAE legal documents
- Multi-language support (English, Arabic)
- Template marketplace with revenue sharing
- Company management with credit allocation

### Frontend Stack
- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + Custom Design System
- **UI Components**: Radix UI + shadcn/ui
- **Routing**: React Router v6
- **State Management**: React Query + Context API
- **Animations**: Framer Motion
- **Real-time**: Supabase Realtime

### Backend Stack
- **Database**: PostgreSQL (Supabase)
- **Auth**: Supabase Auth (JWT)
- **Storage**: Supabase Storage (future)
- **Edge Functions**: Deno Runtime
- **Vector Search**: pgvector extension
- **Real-time**: Supabase Realtime subscriptions

### AI & Integrations
- **AI Models**: OpenAI GPT-4o
- **Research**: Perplexity API
- **Embeddings**: OpenAI text-embedding-ada-002
- **Payments**: Stripe
- **Email**: Resend
- **Webhooks**: Custom webhook system

### Security
- Row-Level Security (RLS) policies
- JWT authentication
- API key management (Supabase secrets)
- Audit logging
- Encryption at rest and in transit

### Performance
